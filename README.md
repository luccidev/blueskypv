# BlueSky Residences — Marketing Site

**Live:** [www.blueskypv.com](https://www.blueskypv.com)

Luxury residential development in Puerto Vallarta, Mexico. Built with Next.js 14, deployed on Vercel.

## Stack
- Next.js 14 (App Router, static export)
- TypeScript
- CSS (global, no framework)

## Development
```bash
npm install
npm run dev
# → localhost:3000
```

## Deploy
Pushes to `main` auto-deploy to Vercel. Connect this repo in the Vercel dashboard and set the domain to `www.blueskypv.com`.

## Unit Data
All unit info lives in `data/units.ts` — edit there to update specs, availability, pricing details.
